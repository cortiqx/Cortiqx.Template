# Cortiqx Templates

Cortiqx Templates is a dynamic event template hub built with React, Vite, and the TanStack framework ecosystem. It offers a premium, customizable collection of event templates designed to streamline your web application development and provide beautifully designed components.

## Tech Stack

- **Framework**: React with Vite
- **Routing & Data**: TanStack Router & TanStack React Query
- **Styling**: Tailwind CSS, class-variance-authority, clsx, tailwind-merge
- **UI Components**: Radix UI, Lucide React
- **Backend/BaaS**: Firebase

## Getting Started

### Prerequisites

Ensure you have Node.js and an appropriate package manager (npm, yarn, pnpm) installed on your system.

### Installation

1. Clone the repository.
2. Install the project dependencies:

```bash
npm install
```

### Running Locally

To start the development server, run:

```bash
npm run dev
```

This will launch the application locally, typically at `http://localhost:5173/` by default.

### Building for Production

To build the project for a production environment, simply run:

```bash
npm run build
```

This generates a production-ready bundle. You can preview it using:

```bash
npm run preview
```

## Features

- **Modern Architecture**: Fast, modular development leveraging the Vite ecosystem.
- **Robust Routing**: Type-safe routing using TanStack Router.
- **Beautiful UI Elements**: High-quality, accessible components powered by Radix UI.
- **Responsive Design**: Mobile-first philosophy ensuring seamless rendering across all devices.
